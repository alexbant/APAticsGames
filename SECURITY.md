# Security Policy

## Supported Versions

The following versions of this project are currently receiving security updates and patches.

| Version                | Supported |
| ---------------------- | --------- |
| Latest Release         | ✅         |
| Previous Major Release | ✅         |
| Older Versions         | ❌         |

If you are using an unsupported version, please upgrade to the latest supported release before reporting security issues.

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please report it responsibly.

### How to Report

Please create a private security report through GitHub's **Security Advisories** feature, or contact the maintainers directly if available.

When reporting a vulnerability, include:

* A clear description of the issue
* Steps to reproduce the vulnerability
* Potential impact
* Proof-of-concept code or screenshots (if applicable)
* Suggested mitigation or fix (optional)

### What to Expect

After receiving your report:

1. We will acknowledge receipt within **72 hours**.
2. We will investigate and validate the issue.
3. We may contact you for additional information.
4. If confirmed, we will work on a fix and coordinate disclosure.
5. Once resolved, a security advisory and patch release may be published.

### Responsible Disclosure

Please do **not** publicly disclose security vulnerabilities until they have been reviewed and addressed. Responsible disclosure helps protect users and allows adequate time to release a fix.

Thank you for helping keep this project and its users safe.
