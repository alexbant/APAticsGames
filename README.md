# APAticGames - Plataforma de Aprendizaje APA Gamificada

## Descripción
APAticGames es una aplicación web interactiva diseñada para enseñar las normas APA (7ª edición) a través de la gamificación. Los usuarios aprenden sobre Citas, Referencias y Formato mediante niveles progresivos, retos diarios y un sistema de recompensas.

## Características Principales
- **3 Módulos de Aprendizaje:** Citas, Referencias y Formato.
- **15 Niveles Interactivos:** Desde fundamentos hasta auditoría avanzada.
- **Sistema de Gamificación:** Puntos, estrellas, rangos y logros desbloqueables.
- **Retos Especiales:** Reto Diario, Jerarquía de Títulos y Desafío Final.
- **Perfil de Usuario:** Estadísticas detalladas, avatares y personalización.
- **Modo Oscuro:** Soporte nativo para reducir la fatiga visual.
- **Diseño Responsivo:** Funciona en escritorio y dispositivos móviles.

## Instalación y Despliegue
1. Clonar el repositorio o descargar el ZIP.
2. Asegurarse de tener un servidor web local (como Live Server en VS Code) para evitar problemas con módulos ES6 y CORS.
3. Configurar `js/firebase-config.js` con tus credenciales de Firebase si deseas usar autenticación en la nube.
4. Abrir `index.html` o `ingresar.html` en el navegador.

## Estructura del Proyecto
- `css/`: Estilos modulares para cada nivel y temas globales.
- `js/`: Lógica del juego, gestión de estado y autenticación.
- `images/`: Recursos gráficos y logos.
- `*.html`: Páginas de la aplicación.

## Tecnologías
- HTML5, CSS3, JavaScript (ES6+)
- Firebase (Autenticación y Firestore)
- FontAwesome (Iconos)
- Google Fonts (Poppins)
